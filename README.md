# Q10 – Gestión Customer Support (Streamlit)

Dashboard en Streamlit para analizar:
- Tickets entrados/cerrados (HubSpot export)
- Gestión por llamadas y WhatsApp
- CSAT (Top-Box 4–5)
- SLA (horas hábiles) y TTR
- Tópicos (TF-IDF + KMeans) estilo “Breeze” usando Asunto + Cuerpo limpio

## Requisitos
- Python 3.10+
- pip

## Instalación
```bash
pip install -r requirements.txt
